Detailed Description

template<size_t SIZE, typename T>
class RwCBuf< SIZE, T >

Rewritable FIFO Ring Buffer Template Class.

Note
There are two operation modes: Rewrite mode and Non-Rewrite mode.
Mode is set by a constructor parameter or by a parameter of the Reset() method.
In Rewrite mode the fill flag is not set.
Parameters
[in]SIZE- buffer size
[in]T- type of buffer items
